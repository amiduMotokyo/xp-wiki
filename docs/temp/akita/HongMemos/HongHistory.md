# 对大鸿年表的梳理

大鸿年表虽然在近七百年间还是空白，但此前的记载足以让我们一窥这个文明的部分特色，并以时间为界记载一些大概的概念。

大鸿的文明围绕着几大主轴进行：
1. 王室继承与君权、祖宗授
2. 礼崩乐坏始末
3. 高和宝船出海、康光陆行舰探索魔界
4. 地府征伐与贺赖政权
5. 塔拉呼那——北方的达摩克里斯之剑和长城建造谭
6. （略小）科举制度的建立
7. 长生妄执：吸血鬼转化仪式还在发力

## 1. 断代

大鸿有着年号制度，但大事年表中同时列出的事太多了，让人反而无法明确记住到底谁开始了下一个年号，所以这里首先整体性捋一下年号更迭。

### a. 传说时代与部落大统一时代（？-b.h.2549）

这是没有年号的世代，第一个年号在此之后，所以我们只能根据第一个年号元年确定这个时代的截止年是天历前2549年。

为方便起见，将天历前写作b.h.（before harmony）

我将天历元年的天历条约译为contract of harmony。和谐条约。

这个时代发生了很多事，包括：

1. 经典尧舜禹，但他们的名字是天官（又称赤帝，受兄禅让）、地官、水官。水官启动了世袭制。
2. 然后这个桀出现了，成汤（没有汤）灭桀，结束传说时代（神话时代），不再有半人半神的领袖（是否意味着天地水三官乃至水官后代有神的特质？）
3. 成王的时代是发展的时代，促进了民族融合，民族自此开始便包罗万象，各地人口膨胀。
4. 然后这个纣王出现了，武王（珷）伐纣，开始了王字旁家族的分封制统治。

### b. 起始之炎（b.h. 2549-b.h. 2400）

这个年代是武王死后，其儿子诵开始的。在这个时代，占据大鸿中原的大型国家【宗】正式建立了，但是别忘记，这是分封制国家的宗主国，仅此而已（*分封给功臣和血亲*）。

宗国的王族姓氏即为宗，因此王的全名是“宗诵”，他建立了始炎年号。

1. 诵王开始了绵延至今的精灵太史制度
2. 平前朝叛乱
3. 疆域扩展至海边
4. 开始礼乐制度：礼的部分区分血缘远近和地位高低，乐的部分则将他们共融

与地球历史不同的是，王的死去不会立刻结束年号。因此诵王死后，始炎年号还在继续，他的儿子继承了王位。

此后直至年号更迭，我们不确定更换过几代王。

1. 百家争鸣
2. 大流星事件导致了年号更迭

### c. 肃运（b.h. 2400-2151）

#### 叩问礼乐

1. b.h. 2339年，值得记录的第一次君主无子离世事件发生，而礼乐制度用族弟入继的方式解决了问题
2. 新君无能导致了百家中部分学派对礼乐制度的反对，甚至得到了部分王室旁支认可

#### 第三类接触

1. b.h. 2227年，无知的中原人发现北边有牧民
2. 在此之后的大多数交流都是贸易交流，包括北边的牧民和近海的渔民

#### 诸王之乱

暂且不去管宗这个家族内部的各种派系是如何形成的，我们知道的就是有两个王在诸王之中崭露头角，两王分别是“珝”和“珻”。

内乱在b.h. 2181年开始，让我们来看看两边的实力对比罢！

1. 珝：支持礼乐制度，王族大部分与主流思想，**绝大部分老祖宗**
2. 珻：对礼乐制度没有好感，部分王族旁支与民间学派，自己家祖宗

鉴于老祖宗真的存在，而且是礼乐制度的食利者，两个人的祖宗在天庭也开打了，有了食利者大多数的支持，珝的祖宗赢了，对珻的祖宗造成了“毁灭性的打击”。

失去神秘先祖力量的珻输了，惨遭灭族。

珻的祖宗将所有力量传给最后的幸存者，珻的女儿，她逃亡海外……（此事在糜家亦有记载）

* 最终结果：怎么又复用始炎了的
