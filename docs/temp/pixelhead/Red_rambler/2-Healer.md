# 矢志医师

## 概述

我之手不执兵戈，我之学不为名利。<br>
暗夜之中，我为一灯；纷争之地，我为中界。<br>
纵刀兵相逼，不夺此志；纵举世疑我，不改此途。

## 必要条件

* <a href="../1-Feldsher" target="_blank">志愿医士</a>达到5级
* <a href="../../../basicJob/Standard-bearer" target="_blank">旗手</a>职业等级达到6级

## 关键属性

* 智力
* 魅力

## 熟练技能

* 防御或闪避
* 医学
* 见闻
* 灵知
* 说服

## 职业特性

通过治疗将非友方单位的HP由负数回复至0时，自身回复3点AP，直到战斗结束前获得1点临时天命，其可以选择退场

## 升级收益

职业等级|奖励能力
:--:|:--:
1|获得战技：教化宣讲
2|升级战技“医者仁心”：救治行动的基础治疗量和使用次数+1<br>获得战技：医者自医
3|获得战技：无关立场<br>升级战技“防患未然”、“战场指挥”：目标不再限定队友，若对中立单位使用，则回合结束时获得1点本场战斗中可用的临时天命
4|获得战技：不可再犯<br>升级特殊战技“急难救援”：若目标不为队友或HP为负，则本次治疗量变为2倍
5|升级战技“教化宣讲”：若即将附加的原印记层数至少为3，则可令那个数值-3，额外附加一层<a href="../../../../../status/mark/#羞愧印记" target="_blank">羞愧印记</a>

!!! info "过量治疗"

    治疗类效果本应回复的HP因目标生命值回复至0或上限而高于实际回复的HP的场合，该效果即为造成了过量治疗

    过量治疗时，治疗类效果本应回复的HP与实际回复的HP的差值即为过量治疗量
