# 漫浪医圣

## 概述

不以险阻退缩，不以荣辱移志。<br>
疲病不辞，死生不避。<br>
若有一日，我忘此誓，天地共弃；若我守此誓至终，不求留名，惟愿后人得见——<br>
在此浊世，曾有人以纯粹之心，行救渡之事。

## 必要条件

* <a href="../2-Healer" target="_blank">矢志医师</a>达到5级
* <a href="../../../basicJob/Standard-bearer" target="_blank">旗手</a>与<a href="../../../basicJob/Guard" target="_blank">卫士</a>职业等级合计达到9级

## 关键属性

* 智力
* 魅力

## 熟练技能

* 防御或闪避
* 抵抗、心理学、魔法抗性任选二
* 医学
* 见闻
* 灵知
* 说服
* 任意话术技能

## 职业特性

通过治疗将非友方单位的HP由负数回复至0时，自身回复3点AP，直到战斗结束前获得2点临时天命，其退场，若其为玩家或BOSS则可以选择不退场

## 升级收益

职业等级|奖励能力
:--:|:--:
1|获得战技：与子同袍<br>升级战技“天命所归”：对于救治行动、防守检定和抗性检定而言，每点天命额外使检定值-5，并为自身回复5点HP或SP
2|升级战技“医者仁心”：救治行动的阈值+8，检定值-4，若奖励骰数量小于医学的奖励骰数量，则获得相当于那个差值的奖励骰<br>升级战技“医者自医”：若那次救治行动造成了过量治疗，则获得X层<a href="../../../../../status/mark/#惩恶印记" target="_blank">惩恶印记</a>，X为过量治疗量（向下取整）
3|升级战技“教化宣讲”：<a href="../../../../../status/mark/#羞愧印记" target="_blank">羞愧印记</a>获得强化：SP小于上限的一半时的敌意行动额外获得层数×2的检定值惩罚；被解除时，每层令持有者失去1AP<br>升级战技“与子同袍”：使用次数+1
4|升级战技“不可再犯”：X为过量治疗量的2倍（向下取整），若<a href="../../../../../status/mark/#惩恶印记" target="_blank">惩恶印记</a>层数在未触发效果的状态下清空，则其拥有者回复2点AP<br>升级“紧急救助”：使用时选择不露出破绽或AP消耗-1
5|升级战技“教化宣讲”：若即将附加的原印记层数至少为2，则可选择X次令那个数值-2（不可为负），额外附加X层<a href="../../../../../status/mark/#羞愧印记" target="_blank">羞愧印记</a>
