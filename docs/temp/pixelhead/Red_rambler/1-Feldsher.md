# 志愿医士

## 概述

我之所往，不问族类，不问信仰，不问阵营。<br>
伤者即我同袍，苦处即我疆场。<br>
凡有呼号，唯以济护为念，不以亲疏易志。

## 必要条件

* 已习得特殊技艺“医学”
* <a href="../../../basicJob/Standard-bearer" target="_blank">旗手</a>职业等级达到2级

## 关键属性

* 智力
* 魅力

## 熟练技能

* 医学
* 见闻
* 灵知

## 职业特性

通过治疗将非友方单位的HP由负数回复至0时，自身直到战斗结束前获得1点临时天命，其可以选择退场

## 升级收益

职业等级|奖励能力
:--:|:--:
1|获得战技：医者仁心<br>获得特殊战技：急难救援
2|获得战技：天命所归
3|升级特殊战技“急难救援”：若目标不为队友或HP为负，则回复1次使用次数
4|升级特殊战技“医者仁心”：救治行动的基础治疗量+2
5|升级特殊战技“急难救援”：每回合首次使用不消耗使用次数与SP，若目标不为队友，则不消耗AP

!!! info "救治行动"

    指效果内容包含回复非自身单位的HP，且不包含会对单位造成伤害的行动

### 奖励能力解释

名称|类型|挂钩属性|SP消耗|效果
:--:|:--:|:--:|:--:|:--
医者仁心|战技|被动|0|造成的所有伤害为非致命伤害，救治行动的目标不再限定队友
急难救援|特殊战技|智力|5|对一格范围内的单位使用，若成功则回复目标（1+差值/10）点HP，若目标不为队友或HP为负，则本次治疗量变为1.5倍
天命所归|战技|被动|0|对于救治行动而言，即使成功也可使用天命（最多修正到大成功）<br>对于救治行动、防守检定和抗性检定而言，每点天命额外使检定值-10
