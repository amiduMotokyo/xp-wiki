# 惹祸精

## 概述

…忘记那些吃空饷的物业吧，适当的朋友费可以让你在碰到那些腌臜破事的时候有一个可以求助的对象。嗯，如果不舍得这些小钱的话，那他们就真的会变得和他们的绰号一样了…
<div align="right">《鬼城商贩手册 卷二》</div>

## 必要条件

* <a href="../1-Street_rat" target="_blank">★街溜子</a>达到5级
* <a href="../../../basicJob/Warrior" target="_blank">武者</a>至少3级
* “迎头痛击”熟练度达到15

## 关键属性

* 力量

## 熟练技能

* 拳，短棍，棍，大锤任选一项
* 防御
* 缴械
* 搬运

## 职业特性

继承武者职业特性，进攻<a href="../../../../status/normal/#晕眩" target="_blank">“晕眩”</a>状态的敌人时，伤害倍率增加0.75

## 升级收益

职业等级|奖励能力
:--:|:--:|:--:
1|获得战技：再试一次<br>升级战技“迎头痛击”：检定阈值+4
2|升级战技“破坚猛袭”：命中部位防具每有10%减伤率，额外造成1点护甲额外生命减损
3|获得战技：完美连段<br>升级战技“再试一次”：使用次数+1
4|升级战技“爆裂回波”：选择额外回复5点HP或3点SP，再根据模式获得对应强化<br>拳：改为获得2层蓄力层数<br>短棍：改为回复2点战技使用次数<br>棍：下次进攻额外造成1点AP减损<br>大锤：下次进攻对主目标以外的目标伤害倍率+0.2
5|升级战技“完美连段”：改为自身与感知范围内的一名其他友方单位获得印记<br>升级战技“迎头痛击”：检定阈值+4，检定值-4
