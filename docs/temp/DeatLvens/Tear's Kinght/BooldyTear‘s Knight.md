# 血泪骑士

## 概述

掌握着血族之力的骑士。但这一行为并未得到市政厅的认可，他们多少会通过其他身份来“合法”的学习鲜血系魔法。

*这些骑士将会远走他乡为了某人的理想而募集同僚，但他们及其那人并不想反抗魔王，只是想给“人类”更加舒适轻松的居所……*

## 必要条件

* 涓泪骑士职业等级达到6级
* 种族为人类血族（可由本社群领导人帮助转化）
* 成为领导人麾下的隐秘骑士（可由本社群领导人册封为骑士，但在王国中不具有其身份证明）

## 关键属性

* 力量
* 魔力
* 相关魔法：<a href="/rules/data/magic/blood/" target="_blank">鲜血系魔法</a>

## 熟练技能

* 选择：剑、长枪、长柄刀、长戟、巨剑、大锤之一作为熟练项
* 选择：弓、弩、轻盾、长盾、巨盾之一作为熟练项
* 防御
* 驯马/驯梦魇二选一
* 战术
* 乔装
* 医术
  
## 职业特性

当自身处于骑乘且拥有“凝血护盾”魔法效果时可将其魔法效果延伸至自身坐骑，护盾总量仍按骑手属性计算。<br>继承涓泪骑士的特性
