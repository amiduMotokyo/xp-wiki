# 第二章：种族

艾克苏普世界群中生活着各式各样的种族社群，在一开始你可以根据自己对它们第一印象进行挑选，不过要注意战役简介中的种族限制。

当你对艾克苏普世界观有了一定的了解之后，角色扮演将会变得更有目的性。通过点击下面的超链接可以跳转到种族及其势力的简介页面。

!!! info "蚁族种族特性的特殊性"
    所有蚁族都有4只手，均无法使用魔法。若选择变体要舍弃一个种族特性时，无法对这两条进行替换和删除。

    “搬运”检定保底提供20点成功难度值的意思是，若搬运检定失败，蚁族也可以提供20点成功难度值；如果搬运检定成功，则可以在20的基础上继续累加差值带来的成功难度值。

    无论是否多一只手作为惯用手，蚁族的4只手仍然可以提供许多便利：你可以最多同时持有1个主武器+3个副武器，总共4个单手武器进行战斗，且交换这些武器在你手中的位置都只需要1行动点。在双持进攻中，你甚至可以在2行动点内同时使用这4个武器发起进攻。不过，这种方式下进攻的上限具有一定的限制：具体来说，那些可以免除双持进攻惩罚的技能，只能对其中最多2个武器生效（即一个主武器和一个副武器的组合）。

    另外，你也可以同时持有2件双手武器进行战斗，或是单手武器与双手武器的组合进行战斗，这为你的武器选择提供了更多的可能性。

    除了这些在战斗层面的便利，蚁族的4只手也可能在rp阶段给你的某些行为带来奖励。请与主持人商量是否可以获得这些奖励。

!!! info "近地悬浮物质"
    腿部防具，力量阈值为5，其他所有属性固定为0，无法升级。

    特殊效果：免疫一切地形效果，若被击中该部位，视为击中躯干。

## 语言

每个文化圈都有自己的官方语言，每个族群都有属于自己的方言。语言的学习可以使你更加方便地与他人沟通。

语言分为官方语言和方言，它们以特殊技艺的形式来展现，挂钩智力属性，拥有熟练度，可以提升水平。

### 官方语言

官方语言是某个文化圈的通用语言，该文化圈使用的文字也与该文化圈的官方语言相对应。

在辨识某文化圈的文字的场合下，如果辨认者不会该文化圈的官方语言，那么会遇到如下惩罚：

* 若没有翻译工具的帮助（比如字典），则无法辨认文字
* 若有翻译工具的帮助，则辨认文字需要用到的检定（通常是学习检定）会获得一枚惩罚骰

在语言沟通的场合下，如果沟通的双方找不到任何一个双方都会的官方语言，那么沟通时会遇到如下惩罚：

* 无法听懂对方讲的话（文字用乱码表示）
* 无法对对方使用任何话术
* 对方的好感度会有所下降

### 方言

方言是指某个文化圈下地方使用的语言。

在语言沟通的场合下，如果沟通双方可以找到一个双方都会的方言，那么沟通时会获得如下奖励：

* 对对方使用话术时获得一枚奖励骰
* 对方的好感度会有所上升

### 母语

每个角色建卡时自带两种语言类特殊技艺，分别为：

* 出生地所属文化圈的官方语言
* 出生地的地方方言

这两种特殊技艺熟练度初始就是30级。

### 语言持有上限

每个角色拥有的语言上限将随着成长而增多，具体如下：

* 每个角色的初始官方语言上限为2个，方言上限为2个
* 每20经历获得1个官方语言上限
* 每10经历获得1个方言上限

### 学习语言

#### 常规学习手段：经历点

角色可以花费经历点来学习外语，这是一种常规的的学习一门语言的方式。

官方语言需要花费2经历点来进行学习。

在学会某文化圈的官方语言后，角色才可以花费1经历点来学习该文化圈下某地区的方言。

#### 常规学习手段：积分

玩家可以花费自己的积分让自己的某个角色学习外语，这也是一种常规的学习一门语言的方式。

官方语言需要花费30积分来进行学习。

在学会某文化圈的官方语言后，角色才可以花费20积分来学习该文化圈下某地区的方言。

#### 团内学习手段

其他学习外语的方式，例如通过学习检定读一本外语入门书等，由主持人自行确定。

以下列出几种参考：

* 长休时，由精通语言者（对应语言熟练度至少25级，无需获得“传授”特殊技艺）教授学习者日常用语：消耗双方3长休行动点，学习者过学习检定，如果过了，获得1级无法升级的对应语言；如果没过，也获得1级无法升级的对应语言，但使用此技能检定时获得一枚惩罚骰
* 去商店购买词典，长休时进行学习：消耗1长休行动点，进行学习检定，根据检定成果累积学习进度，学习进度满了之后可以学会对应语言，具体如何设置学习进度由TG根据语言的学习难度来确定
* 另外，词典也可以在语言检定时配套使用，获得一些奖励，具体由TG来指定
* 手语：语言不通时，一个成功的乔装检定可以让你和对方进行大概的沟通

### 使用语言检定的场合

默认情况下，角色将使用官方语言对应的文字，说话时使用的是母语，即带有自家方言口音的官方语言。在这些场合下都无需进行语言检定。

以下列出了几种需要进行语言检定，但检定难度不高（例如-20）的场合：

* 辨识非母语的文字
* 听懂非母语的官方语言
* 听懂非母语的方言
* 听懂有挑战性的母语语言（例如古语、艺术作品）
* 辨识有挑战性的母语文字（例如古语、艺术作品）

当对应的语言能力熟练度达到15级后，以上场合就都不需要再过检定了。但这也意味着以后遇到这种场合也无法再增长经验了。

以下列出了几种需要进行语言检定，但检定难度一般（例如0）的场合：

* 讲有挑战性的母语（例如古语、艺术作品）
* 写有挑战性的母语（例如古语、艺术作品）
* 讲非母语的方言
* 讲非母语的官方语言
* 写非母语的文字
* 听懂有挑战性的非母语语言（例如古语、艺术作品）
* 辨识有挑战性的非母语文字（例如古语、艺术作品）

当对应的语言能力熟练度达到30级后，以上场合就都不需要再过检定了。但这也意味着以后遇到这种场合也无法再增长经验了。

以下列出了几种需要进行语言检定，但检定难度较高（例如40）的场合：

* 讲有挑战性的非母语（例如古语、艺术作品）
* 写有挑战性的非母语（例如古语、艺术作品）

如果检定成功，在接下来的一段语言沟通活动中可以获得方言带来的奖励，或是不会受到官方语言缺位的惩罚。如果检定失败，则效果相反。

特别情况下，例如沟通双方均用非母语进行对话，则只需其中一方检定成功，就算双方都检定成功。

#### 方言

基本准则是（一片地区+一种官方语言）对应一个方言。由于地区众多，因此不方便使用表格的形式列全。

特殊情况下，一片地区由2个文明共治（例如贝塞克顿、菲布里底），则这2个文明在这片地区各执一种方言（例如贝塞克顿精灵方言与贝塞克顿人类方言是两种不同的方言）。
