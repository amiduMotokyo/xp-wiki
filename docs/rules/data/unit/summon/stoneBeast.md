# 石兽

## 形象

暂缺

## 简介

受到东方大鸿国的影响，法师们开始将塑型土构造成石兽的模样，尤其是在爪部下了很大的功夫。石兽的移动力和防御力较为薄弱，但它们被激活后的一爪威力无穷。
