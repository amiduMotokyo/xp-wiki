# 鸡蛇兽

## 形象

暂缺

## 简介

下层世界家畜，肉质鲜美，但脾气暴躁，住在农村的多多少少都尝过它们的大嘴巴子。因为意外地很受三界人民喜爱，所以法师们也开始研究它们的习性，做出了同名的召唤物。据说鸡蛇兽还有石化的能力，不过用塑型土捏出来的自然是没有了。
