# 通用坐骑技能

原则上来说，隶属于某种坐骑类型分支下的技能只能适用于该种类的坐骑，不过如果画风合适，或者技能中有特殊说明，主持人也可以考虑允许跨坐骑种类使用。

注：`坐骑检定`指坐骑的性能属性+骑乘者的驯兽熟练度；`驯兽检定`指骑乘者的智力+骑乘者的驯兽熟练度。两者均算作白板数值，且共享`驯兽检定`的奖励和好处。
