# 武器及其平均性能表

## 升级公式

每升一级可以选择让：

* 力量阈值-5（最低5，最高60）
* 武器威力+1（最多+10，最低-4）
* 持有“有属性加成”标签的武器，获得2点自由分配的装备属性加成（每种属性最多+6，最低-6，反向兑换规则略有不同，见后文）
* 近战武器无法加射程；若为远程武器，获得1点射程（最高+5，最低-5）
* 没有施法范围的武器（默认1施法范围），可以花费3等级让施法范围变为2，此后每一级可以获得1施法范围加成（最高+8，最低-8）
* 每类武器独有的升级方式

每件武器上限15级，每提升1级都会让提升等级的代价增大，且不能降级

不过，你可以通过反向操作等量的上述值来把好处匀给其他属性，代价是消耗量变为2倍（比如通过增加10点力量阈值来让武器威力+1）。此外，通过减少属性来兑换好处时，单属性-6才能换来相当于1武器等级的其他好处，且这种兑换方式限定力量、敏捷与体质

此外，武器在升至5、7、9、11、13、15级时会迎来质变，这会为武器提供若干个大小印痕位并同时印刻对应的印痕，具体规则为：5级、7级时，分别解锁一个小印痕；9级、11级时，分别解锁一个大印痕；13级时，解锁2个小印痕；15级时，解锁2个大印痕

质变武器会为使用者带来多样化的好处，但质变后的武器使用难度更高（若不能达到武器要求的经历点与熟练度，则使用时视为把武器降级到能达到要求的最大等级），同样的，也只有那些技艺精湛的铁匠才能开启高等级印痕位并印刻。

重铸武器需要在相应质变等级所需求的铁匠处才能重铸，且每次只能将至多一个印痕转化成同级别的另一个，若要转化更多的印痕则需要对其中的每个额外支付重铸价格的一半

### 投掷（飞镖）系

!!! info "投掷系武器的共通特殊效果"
    该类武器不可在铁匠铺中进行升级，只能直接购买对应等级的武器。挂钩属性都为力量或者敏捷，攻击范围都为5+（力量或敏捷-力量阈值）/10。

    特别地，你无法通过“锻造检定”，或者在铁匠铺中，对该类武器进行“补充弹药”操作。

    这意味着，你需要频繁地补充投掷系武器的弹药，并根据你的经济和经历点情况购入相应等级的投掷系武器。好在购入这类武器时通常一次性会给很多。

    如果要售出该类武器，则售出的价格按照（一般武器价格/批发弹药量×剩余弹药量）来算。

!!! info "可投掷的物件"
    当你使用投掷系武器进攻时，自然可以用投掷检定。除此以外，投掷以下几种物件，也可以使用投掷检定，也可以适用投掷系升级技能和远程进攻通用技能，攻击范围都为5+（力量或敏捷-力量阈值）/10。

    * 可单手持用的近战武器，挂钩属性为原武器的挂钩属性，不可适用原武器的升级技能和通用技能。结算完伤害后，武器会丢弃在目标地块或目标所在地块上。
    * 可单手持用的远程武器，与近战武器的区别是，武器威力固定为3。
    * 道具，挂钩属性为力量或者敏捷，力量阈值固定为0，武器威力固定为3，投掷时会露出破绽（视为使用道具）。结算完伤害后，根据道具描述，由主持人自行决定是否触发道具效果。

### 火器系

!!! info "火器系武器的共通特殊效果"
    火器是相当先进的远程射击武器，其击发的弹药速度远大于其他远程武器，令人难以反应，因此目标只能用防御对抗，且对抗阈值减半。相应地，火器无法进行类似于抛射的弹道复杂的攻击，因此只能进行直射。

    火器的装弹操作非常繁琐，因此都需要消耗大量行动点，且进行装弹时会被视为一次<a href="../../../V4.x rules/Extra/BulletTime" target="_blank">破绽</a>。

    若角色在持用火器的情况下，受到<a href="../../status/normal/#潮湿" target="_blank">“潮湿”</a>状态，或者进入“水”单元格后，火器会受潮，本场战斗将无法继续使用。战斗结束后恢复正常。

    由于火器发射时会发出极大的声响，因此即使是在隐身状态，使用火器进攻时，也强制破影一击。
