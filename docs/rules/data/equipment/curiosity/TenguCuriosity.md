# 神器与套装（天狗国）

神器中的属性增益默认均可超上限，但若其与角色属性值相关，则同名属性效果不叠加，取较高者。

凑套装效果时，统计的是防具、主武器和副武器中装备穿戴的情况，并不统计道具栏中装备的持有情况。

## 传世套装

天狗国的传世套装为过去五位现世神或现世神的亲属生前的披挂。若能集齐全套，配合五位现世神留下的神明遗物，可开启对应神明的完全凭依。

完全凭依不需要凭依者掌握普通神凭的方法。相比普通神凭，完全凭依的区别在于：持续时间为整场战斗，自动掌握所有对应神明旗下的阴阳术与神凭能力，冷却时间为一次长休。
