# 血爪

## 概述

相对于影羽的状况，他们在评选中的战斗能力较为的出色，便被选择为了正面作战人员。主要参与小规模集团作战的先锋，为工会执行更加激烈的战事。他们以嗜血著称，经过特殊的训练后精通于对已经受伤的敌人不断的追击致其于死地。<br>“血腥的气味总是让我感到兴奋难耐，不知道下一次是什么时候开始。”

## 必要条件

*武者职业达到5级

## 关键属性

* 敏捷

## 熟练技能

* 选择一种非拳套、棒槌、杖、鞭子、棍、大锤、巨盾系近战武器
* 战术

## 职业特性

继承武者所有特性。<br>获得战技：工会支援
