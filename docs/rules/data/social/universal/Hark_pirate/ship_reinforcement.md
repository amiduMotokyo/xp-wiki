# 舰船支援

## 什么是舰船支援

当有舰船在战场外对战场内提供一定程度的支援时，受到支援的那一方即受到舰船支援。<br>由于过大的体积和场地原因，船只通常不能直接进入战斗地图作战，这时它就会在战场附近提供一定的支援，这种支援包括火力打击和其他支援。<br>舰船由船只本体与船只船备构成。

## 船只本体

### 属性

舰船有着以下几种属性：<br>类型：这里的名称指船只的标准型号，不同类型的船只有着不同的属性。<br>大小：虽然舰船不会直接进入战斗，大小并没有确切的规定，但是可以通过这点来判断未知型号船只的其他属性。<br>舰炮上限：规定了一艘船可以武装多少门火炮，火炮等级和数量将在火力打击中直观体现。<br>支援系统上限：规定了一艘船可以武装多少支援系统，除火力打击以外其他的支援方式并称为支援系统。<br>船员上限：规定了一艘舰船最多可以拥有多少船员，火炮和支援系统都需要对应数量的船员来操纵。<br>特殊效果：这艘船拥有什么特别的能力。

## 船只船备

船只船备分为舰炮与支援系统构成<br>当可用船员达到需求时才可启用系统，启用系统后船员将会被占用不再可用

### 舰炮

舰炮即装备在舰船上的火炮，可以在战斗中支援战场，船炮需要玩家操控或者特定技能指挥才能开火，且开火后需要一回合冷却时间，每门炮独立计算冷却时间。

名称：指舰炮的类型，不同类型的舰炮有着不同的属性。<br>操纵船员数：操纵这门炮所需的船员数量<br>操纵难度：在玩家操控时，成功操纵舰炮所需额外鉴定难度<br>攻击范围：舰炮攻击的伤害范围<br>攻击阈值：舰炮攻击时提供的鉴定阈值，默认为npc攻击阈值<br>威力：舰炮的攻击威力<br>特殊效果：这门舰炮拥有什么特别的能力

### 支援系统

除了最直接的火炮轰击，舰船还可以拥有其他支援战场的手段，这些手段被称为支援系统。

名称：该支援系统的名称<br>操纵船员数：操纵这个支援系统所需的船员数量<br>类型：该支援系统触发类型<br>特殊效果：这个支援系统拥有的特别的能力
