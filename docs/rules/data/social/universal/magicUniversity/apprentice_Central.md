# 法师候补生

## 概述

法师候补生是指在中央联合魔法大学授权下，拥有成为正式法师资格的魔法入门研修者。相比苏尔伍德第二魔法大学的魔法学徒，法师候补生更倾向于按部就班地学习官方教科书上的魔法，并熟记它们的魔操套路。

## 必要条件

* 贵族出生，或拥有至少10点经历点且在贵族圈有熟人推荐
* 魔力属性>=30
* 选择1个<a href="../magicUniversity_Central/#学院信息" target="_blank">学院</a>作为主修学院（本学院）
* 选择1个<a href="../magicUniversity_Central/#学生自治组织信息" target="_blank">学生自治组织</a>参加，提供熟练技能
* 无法选择<a href="../apprentice_Second" target="_blank">魔法学徒</a>职业，但可以通过办理转校（需符合对方职业必要条件）把本职业等级替换为对方职业等级

## 关键属性

* 魔力

## 熟练技能

* 魔法感知
* 魔法抗性
* 所选<a href="../magicUniversity_Central/#学生自治组织信息" target="_blank">学生自治组织</a>的熟练技能
  
## 职业特性

由于难以彻底排查是否存在禁忌魔法，故禁止使用“典籍类”魔法武器。

获得1本记载了<a href="../magicUniversity_Central/#准学学派" target="_blank">准学学派</a>中主修学院魔法大类的所有初级魔法的魔法教科书（不可售卖）。

在长休中<a href="/rules/V4.x rules/8·magic/#通过魔法书学习魔法" target="_blank">通过这本魔法教科书学习魔法</a>时，消耗的长休行动点-1。

在战斗中想<a href="/rules/V4.x rules/8·magic/#通过魔法书使用魔法" target="_blank">通过持用这本魔法教科书使用魔法</a>获得好处时，由于翻找困难，耗费的行动点+1，但可以将“若已掌握了该魔法”的奖励替换为“检定值-4以及阈值+8”。
