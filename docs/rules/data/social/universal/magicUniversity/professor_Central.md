# 授权教授

## 概述

授权教授是指在中央联合魔法大学授权下，拥有正式教授编制的施法大师。相比苏尔伍德第二魔法大学的魔法教授，授权教授能够熟练查阅魔法教科书上的所有魔法，并在战斗中更加高效地使用它们。

## 必要条件

* 贵族出生，或拥有至少25点经历点且在贵族圈有熟人推荐
* <a href="../postgraduate_Central" target="_blank">受训法师</a>职业等级达到5级
* 某魔法流派等级达到3级及以上
* 已获取“传授”的特殊技艺
* 选择2个<a href="../magicUniversity_Central/#学生自治组织信息" target="_blank">学生自治组织</a>作为指导老师，提供熟练技能
* 如果选择秘密结社指导，熟练项可替换为：任选2种话术
* 无法选择<a href="../professor_Second" target="_blank">魔法教授</a>职业，但可以通过办理转校（需符合对方职业必要条件，所选学院不可变）把本职业等级替换为对方职业等级
  
## 关键属性

* 魔力
* 智力

## 熟练技能

* 魔法感知
* 魔法抗性
* 任意魔法武器（除“典籍类”魔法武器）
* 灵知
* 见闻
* 传授
* 所选<a href="../magicUniversity_Central/#学生自治组织信息" target="_blank">学生自治组织</a>的熟练技能

## 职业特性

由于难以彻底排查是否存在禁忌魔法，故禁止使用“典籍类”魔法武器。

获得2本记载了<a href="../magicUniversity_Central/#准学学派" target="_blank">准学学派</a>中某1魔法大类的所有魔法的魔法教科书（不可售卖）。

在长休中<a href="/rules/V4.x rules/8·magic/#通过魔法书学习魔法" target="_blank">通过魔法教科书学习魔法</a>时，消耗的长休行动点-1。

在战斗中想<a href="/rules/V4.x rules/8·magic/#通过魔法书使用魔法" target="_blank">通过持用魔法教科书使用魔法</a>获得好处时，将“若已掌握了该魔法”的奖励替换为“检定值-8以及阈值+16”。
