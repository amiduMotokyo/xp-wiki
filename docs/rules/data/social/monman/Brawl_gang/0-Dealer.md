# 小巷贩商

小巷贩商是非战斗职业，无法作为主职，只能作为兼职。

## 概述

“想来点好货吗，我敢打赌你没见过这么纯的…不想？嘁…”

## 关键属性

* 智力

## 熟练技能

* 制药
* 销售
* 灵知

## 升级收益

职业等级|奖励能力
:--:|:--:
1|在药品加工表中新增“可食用冰”，进行“兜售货物”时，若目标为“可食用冰”，则无法通过销售获得额外利润且需消耗0.5长休行动点
2|获得长休行动：原料采购
3|获得战技：推销话术
4|获得战技：吸嗨技巧<br>升级长休行动“原料提纯”：单个素材上限变为+5
5|将“恐吓”加入熟练技能<br>升级长休行动“制成药品”：制成“可食用冰”时，可消耗两份等级相同的制药素材，若此做，制成两份“可食用冰”
6|获得战技：隐蔽内袋<br>升级战技“吸嗨技巧”：每轮首次使用“可食用冰”时不会露出破绽且消耗AP-1
7|获得长休行动：提纯巧思
8|升级战技“吸嗨技巧”：全行动阈值再+X<br>升级战技“推销话术”：对敌人使用话术时，若“可食用冰”效果生效中，则附加的印记层数变为（1+差值因子）层

### 新药品

#### 可食用冰（纯度等级X）
* 素材品质要求：+X制药素材，不同纯度的可食用冰共享（玩家数量×2）的携带上限
* 效果：一轮内全行动检定值-X，获得一枚奖励骰；效果结束后进入<a href="../../../../status/normal/#晕眩" target="_blank">“晕眩”</a>状态，一轮内全行动阈值-X
* 售价：100X通

### 新长休行动

* 原料采购【0.5长休行动点】：若长休地点位于城镇内，可以进行原料采购行动，消耗50X通，购买合计+X的，仅可用于制成“可食用冰”制药素材，单个素材上限为+3
* 提纯巧思【1长休行动点】：进行一次灵知检定，若成功，则下次制成“可食用冰”时，纯度等级增加1+（差值因子/20向上取整），纯度可以超过10
