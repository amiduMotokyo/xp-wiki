# 狩猎大巧思

狩猎大巧思是一种战斗技巧，而非战斗职业，无法作为主职，只能作为兼职。

## 概述

俺寻思不够大的家伙应该可以主动变大，俺寻思大个家伙应该都有一个弱点给俺们打打，俺寻思没什么装备是俺们扒不下来的，俺寻思打掉弱点后对面应该会爆点好东西给俺们。

想到这里，巨兽猎手们忍不住高唱起了《又大又好之歌》。

## 必要条件

* 拥有任意巨兽猎手社群职业的职业等级

## 关键属性

* 无

## 熟练技能

* 复制任意巨兽猎手社群2星职业的熟练技能
