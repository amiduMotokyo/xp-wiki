# 追迹猎手

## 概述

追迹猎手秉持“对面不想正面打架，那就胁迫他们正面打架”的原则，通过使用香啊、很香啊的香料给对面打上标记，来让他们失去隐蔽的效果。与此同时布置陷阱也是他们的拿手好戏，当对面不得不正面冲过来的时候，脚下的陷阱会让他们再吃一次亏。

## 必要条件

* 已获取“采集”的特殊技艺
* <a href="../../basicJob/citizen" target="_blank">魔族公民</a>达到1级
* <a href="../../../basicJob/Sentinel" target="_blank">哨兵</a>职业等级达到3级

* 敏捷
* 智力

## 熟练技能

* 任意远程武器
* 侦查或感知二选一
* 潜行
* 陷阱技术
* 采集
  
## 职业特性

继承哨兵的职业特性。

战斗开始前，可以在地图上，敌方索敌范围（对隐形）外的任意位置预先布置好一个免费的任意1×1陷阱或已有的2×2陷阱。

额外的魂源获取方式：

* 每个敌人限定一次，当他触发了你布置的陷阱效果后，你立刻获得相当于他经历点数值的魂源
* 你每施加1层<a href="../../../../status/mark/#气息印记" target="_blank">“气息印记”</a>，你获得3点魂源

## 特殊技艺表

### 陷阱技术

* 描述：制造陷阱，或是验证陷阱效果时需要用到的检定。
* 挂钩属性：智力
* 战斗效果：
* 临阵造雷：消耗3点行动点，露出一个破绽，进行一次制造陷阱的长休行动
* 布置陷阱：消耗3点行动点，露出一个破绽，在自己或周围1格范围内布置一个陷阱。
* 长休效果：
* 制造陷阱【0.5长休行动点】：根据陷阱制造表，消耗对应品质的素材，进行陷阱技术检定，若成功，则制成对应陷阱。

#### 触发陷阱流程

一个陷阱一旦被布置好，即刻进入“隐形”状态，且仅可被识破隐形中与“侦查”检定有关的手段识破（对抗潜行改为对抗布置者的陷阱技术）。被识破后可以在1范围内花费2行动点通过妙手检定解除。若要这么做，敌人需要用妙手检定对抗陷阱布置者的“陷阱技术”检定，若成功，则可拆除陷阱。

对于一个已经布置好的陷阱，没有碰撞体积，默认占地1×1，激活后消失。当其上存在敌人时激活，敌人需要用闪避检定对抗陷阱布置者的“陷阱技术”检定，若失败，则触发陷阱效果。若成功，则无视发生，陷阱本行动轮在该敌人的回合内失效。
