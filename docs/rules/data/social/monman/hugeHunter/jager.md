# 巨兽猎兵

## 概述

巨兽猎兵是扛着又长又大的武器去和巨兽正面肉搏的职业，小型武器他们看不上。他们最喜欢的作战方式是爬到一个又强又大的目标肩膀上给他重重地来一下，然后再来一个英雄式的落地。

## 必要条件

* <a href="../../basicJob/citizen" target="_blank">魔族公民</a>达到1级
* <a href="../../../basicJob/Warrior" target="_blank">武者</a>职业等级达到3级

## 关键属性

* 体质
* 力量

## 熟练技能

* 任意一样“长柄武器”或“重型武器”技能
* 攀爬
* 跳跃
* 缴械
  
## 职业特性

继承武者的职业特性。自带“巨兽杀手”技能，且此技能对“重型武器”也生效。

!!! success "巨兽杀手（长柄武器10级通用技能）"
    对占地超过1×1的单位造成的伤害总伤倍率+0.5倍

额外的魂源获取方式：

* 每个敌人限定一次，每当你对其施展与攀爬检定有关的能力后，你立刻获得相当于他经历点数值的魂源
* 每个敌人限定一次，若亲手击败一个占地超过1×1敌人，你立刻获得相当于他经历点数值×2的魂源

## 升级收益

职业等级|奖励能力
:--:|:--:
1|获得战技：登顶一击
2|获得战技：强势攀爬
3|将“抵抗”、“防御”、“闪避”三选一加入巨兽猎兵的熟练技能中<br>战技“登顶一击”升级：若目标体型（占地格数）比你大，目标对抗你登顶一击时的所有检定额外获得一枚惩罚骰
4|获得战技：补一脚
5|战技“强势攀爬”升级：消耗的行动点-1<br>战技“登顶一击”升级：若你登顶一击的进攻对抗差值因子大于等于3，则目标强制<a href="../../../../status/normal/#倒地" target="_blank">“倒地”</a>

奖励能力解释：

名称|类型|挂钩属性|SP消耗|效果
:--:|:--:|:--:|:--:|:--:
登顶一击|战技|触发|0|每行动轮每个目标限一次，允许登上体型（占地格数）比你大或相同的单位：在目标1格范围内花费2行动点，用你的攀爬检定对抗对方的抵抗检定，若成功，即可登上目标并施展登顶一击
强势攀爬|战技|触发|5|每行动轮限2次，选取4格范围内目标（也可以是需要攀爬的地形），花费3行动点，立刻向该目标进行免行动点的移动，过程中无视目标的借机与威压以及地形对移动的惩罚，若你因此进入了该目标周围1格，可立刻免行动点对其执行一次与攀爬检定相关的行动
补一脚|战技|触发|3|你在“翻越”行动与“登顶一击”落地后选择触发，立刻免行动点对目标进行一次“推搡”（缴械相关，不管之前使用过几次）或“飞踹”（跳跃相关，不管你有没有学会）行动，且此行动可以合并到任何敌意行动当中视为一个行动

!!! success "登顶一击"
    当登上目标后，你可以免行动点立刻对目标进行一次进攻，本次进攻可以免费瞄准目标除腿部以外的部位，无视长距离武器惩罚与近战接触惩罚，无视目标一半的护甲值，但无法获得战术奖励；目标只能用防御进行对抗，且无法获得战术奖励。

    如果本次进攻成功或使目标<a href="../../../../status/normal/#濒死" target="_blank">“濒死”</a>，你立刻选取目标周围1格内的一个空单元格作为你的落地点，平安落地。

    如果本次进攻失败，你会被目标甩下，落地点为目标周围1格（由目标选择），受到目标每一点身高等价高度（正常角色默认为1）1d5点落地固定伤害。
