# 海族智者

## 概述

拥有超过人智的头脑，王子的直系下属。使用工具与武器的水平与智慧生物几乎无异，并研究新的进化方式。

## 必要条件

* 必须是开智海族
* <a href="../umiumeshikikan" target="_blank">海族指挥使</a>职业等级达到5级

## 关键属性

* 力量
* 敏捷
* 智力

## 熟练技能

* 海族指挥使职业中选择的武器
* 搬运
* 缴械
* 学习
* 灵感
  
## 职业特性

允许使用一种武器，允许提升武器的熟练度，但升级所需经验值翻倍，且只能习得选定武器的5、15级武器类型技能与10、20级武器通用技能
