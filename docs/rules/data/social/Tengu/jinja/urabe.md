# 卜部

卜部是一种纸符使用技巧，而非战斗职业，无法作为主职，只能作为兼职。

## 概述

卜部是神社体系中专职占卜的技术职员。早期占卜通常使用龟甲，而当今的占卜更倾向于使用便于携带的纸符。

## 必要条件

* <a href="../iwai_miko" target="_blank">祝/巫女</a>职业等级达到5级

## 关键属性

* 魔力

## 熟练技能

* 纸符
* 魔法感知
* 魔法抗性、闪避、防御三选一

## 升级收益

3|同第二级
4|同第二级
5|同第二级

奖励能力解释：

名称|类型|挂钩属性|SP消耗|效果
:--:|:--:|:--:|:--:|:--:
纸符开光|战技|被动|0|允许你将主职熟练技能中的任一武器技能替换为“纸符”<br>允许将天狗国阴阳术的第一种<a href="/rules/V4.x rules/8·magic/#仪式之法" target="_blank">仪式之法</a>素材（各种神签）当做纸符武器的弹药使用（若没有纸符武器，视为1级的标杆纸符）
赐福纸符|战技|被动|0|你使用纸符进攻时，允许如同“祭祀”特殊技艺的战斗效果“神罚”一样，根据信奉神明不同，附加不同的效果
符术|战技|被动|0|当某一天狗国阴阳术准备完成后，你可以选择将其存入你的一张纸符中，在之后的1行动轮内的任意时刻，选择施法范围内的目标，将其贴在目标朝向你的身体部位上<br>若超过1行动轮还没有张贴，则纸符上的阴阳术效果失效

武运昌隆|战技|被动|0|战斗开始时，你可以选择消耗1张纸符，进行一次有1枚奖励骰的幸运检定，若成功，获得“检定差值”的点数，每1点点数可用于使本场战斗中的任意检定的阈值增加/减少1点（允许在检定后进行变动），对于单次检定而言，最多令其变动10点

!!! success "符术细则"
    被携带在纸符上的阴阳术效果会正常作用于被张贴的目标，且不再拥有弱点部位。想要解除符术，有如下几种途径：
     
    1、来到目标1范围内、符术存在身体部位的方向上，花费2行动点，用妙手检定对抗符术施加者的魔法抗性检定，若成功，即可解除符术。
     
    2、在符术存在身体部位的方向上，对目标进行进攻，此时如同传统规则，即击中仪式之法的弱点一般，进行解除判定。如果该进攻造成至少5点火焰伤害，则可无视对抗立刻解除该符术。
     
    示意图如下所示：

    [![petq88H.png](https://s41.ax1x.com/2026/04/05/petq88H.png)](https://imgchr.com/i/petq88H)

    特别地，对于中了符术的目标而言，如果符术存在的身体部位是你的正面或侧面，那么你自己就可以进行妙手检定对抗进行解除；如果符术存在的身体部位是你的背面，你只能靠其他人进行解除。

    如果中了符术的目标陷入了<a href="../../../../status/normal/#潮湿" target="_blank">“潮湿”</a>状态，则在解除对抗时，符术施加者的魔法抗性检定获得一枚惩罚骰。
