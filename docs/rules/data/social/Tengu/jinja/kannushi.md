# 神主

## 概述

神主是神社体系中一社的领导者，是神社祭祀事务的首席执行者。虽说大社的神主一般由大神的直系后裔担任，但分社的领导者则可以从那些信仰极其坚定者当中选举出来。

## 必要条件

* 宣誓侍奉某一位大神
* <a href="../negi_omiko" target="_blank">祢宜/御巫女</a>职业等级达到5级
* <a href="../../../basicJob/Standard-bearer" target="_blank">旗手</a>职业等级达到5级
* 若供奉的大神是锻造大神，额外需要：已习得“锻造”特殊技艺

* 杖
* 三种话术（说服、哄骗、恐吓）任选其一（若供奉的大神是稻荷大神，此处固定为“说服”）
* 魔法感知
* 魔法抗性
* 祭祀
* 根据供奉大神，获得不同的熟练技能

## 职业特性

继承旗手的职业特性。

随身携带不计量的空白神签。

你在准备天狗国阴阳术的仪式时，允许你照常执行任何行动。

施展天狗国阴阳术时，去除“通过各种手段减少准备时间，最多只能将准备时间减少到2回合”的限制。

你的主武器为杖系武器的场合下，若你在杖系武器的“携带模式”下进入天狗国阴阳术准备仪式，准备过程中每一次执行“祭祀”的战斗效果“大神赐福”都会额外使所有正在准备的阴阳术的准备时间减少半行动轮；若你在杖系武器的“固定模式”下进入天狗国阴阳术准备仪式，准备过程中除非你的杖系武器被缴械，你不会因被进攻到弱点而被打断施法，且你已经施加的所有阴阳术都无法被外力卸除。
