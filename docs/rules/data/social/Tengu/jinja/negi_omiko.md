# 祢宜/御巫女

## 概述

祢宜（男性）或者御巫女（女性）是某位大神的坚定信徒，他们被允许主持神社中一些重要的祭祀活动，能够获得大神更多的青睐。

## 必要条件

* 宣誓侍奉某一位大神
* <a href="../iwai_miko" target="_blank">祝/巫女</a>职业等级达到5级
* 拥有至少8个经历点

## 关键属性

* 魔力

## 熟练技能

* 杖
* 三种话术（说服、哄骗、恐吓）任选其一
* 魔法感知
* 魔法抗性
* 祭祀
  
## 职业特性

继承祝/巫女的职业特性。

你在准备仪式之法的仪式时，允许你照常使用话术。

你的主武器为杖系武器的场合下，若你在杖系武器的“携带模式”下施展仪式之法准备仪式，准备过程中每一次执行“祭祀”的战斗效果“大神赐福”都会额外使某一仪式之法的准备时间减少半行动轮（最多减少到半行动轮）；若你在杖系武器的“固定模式”下施展仪式之法准备仪式，准备过程中除非你的杖系武器被缴械，你不会因被进攻到弱点而被打断施法。
