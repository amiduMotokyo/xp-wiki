# 天狗国家传特性

天狗国的每个大家族会为其家族成员提供家传特性，也就是你的第二个种族特性。

理论上来说，由于“苗字”是可以互相赠送的，“氏”也是可以通过一定努力挣得的，因此并不存在某个“氏”或者某个“苗字”独属与某一个种族的说法。不过，领有这些“氏”和“苗字”的种族确实也存在一些分布规律，可以在画风层面进行参考。

## 大神后裔

在选择这些家族时，应当尽量贴合其种族分布，原因不仅在于画风，而且还在于游戏性：不同大神旗下“阴阳术”所消耗的第二种素材为赐福鸦羽/狐毛/刚毛，如果你想充分发挥这类家传特性的优势，选择加入<a href="/rules/data/social/Tengu/jinja/jinja/" target="_blank">“神社”</a>社群，那么你将会获得自行生产这些素材的能力，而只有鸦天狗/狐天狗/鲸天狗才能产出对应的赐福鸦羽/狐毛/刚毛。如果你的种族没有和大神后裔当中标注的多数种族对应，那么你产出的第二种素材将会出现不配对的情况。

## 其他家族

如果你的家族不是以上中的任何一个，你可以自创你的家传特性，请和主持人商议后决定；若想要沿用某一“苗字”家族的特性，可以设定为其分家。
