    天使与天界意见征集稿

    写作目的

    鉴于现阶段天界设定相当不足，而天使社会全体作为重要的文化和政治影响因素存在于整个西陆地区，希望以此文件为基础，描绘西陆人民普遍印象中的天使形象。

    后续延伸至天使社会中存在的教化材料，表达一个前往天界首都塞纳的观光客所能观察到的天界社会形态，乃至天界当局的社会组织形态对天使群体的政治宣传和基础教育内容。

    并兼顾"一名天使的日常"，讲述一名普通天使从出生到成人可能会遭遇的种种情况。

    上层世界被分配到最少份量的魔法粒子，生物整体进化风格呈现懒惰化。白天明亮，黑夜里可以望见一望无际的夜空和闪烁其中的星星。“明亮而单调”

    b. 灵魂循环机制与智慧生命的形成

    • 【信仰的基础】

    ……灵魂会在魔法粒子的驱动下进行短时间的爆发，从而自发实现单程“穿界”，向魔法粒子的两极移动。强烈的意志能够一定程度上影响该移动的方向。

    ……穿界过程中，由于魔法粒子浓度急剧改变，灵魂部分意识失散，从而失忆。

    • 【天界土著的来源】

    原初世界的住民是一种人形生物，能够驱使魔法粒子对自己的身体进行改造（包括翅膀、鱼鳍的生成），文明程度止于原始部落，对魔法粒子依赖性强，无法脱离。当他们被世界的撕裂丢到上层世界时，因为极低的魔法粒子水平而陷入危机。

    天界居民打通了上中两层空间的链接，长期通过高维空间作为传递介质，从中层的【同胞】身上获取（魔法粒子提供的）能量，【维持】住了原本的能力，建立了繁荣的文明。

    • 【信仰的诞生】

    时间流逝，中层沧海桑田，能量供给随记忆一同减少。天界居民首先实现了穿界的创举，然而发觉【同胞】已然不再，中层居民退化，天界居民顺势建立【宗教】。“以信仰的方式重新建立了中层世界到上层世界的能量供给桥梁”。维持高度文明的同时，确保自身能力不会衰退。

    • 【后来者之道】

    在中层世界死后升入上层世界的灵魂为了能让自己在【转生前】过上好日子，也开始学习天使，在中层世界创立宗教，从而升格为神。

    • 【神之道】

    神会用自己的方式回应信徒的请求，给予他们能量以换取持续的信奉。【仪式之法的工作原理】

    f. 补充资料：灵魂的诞生与灵魂的大循环

    • 【灵魂的诞生】

    大脑，是智慧生物特有的思维与意识器官。当新的智慧生物生命从母胎中诞生时，大脑会且仅会产生唯一的【意识原体】，在此后发育成熟为所谓的【意识】。而【意识】将借助魔法粒子的力量汇聚，成为【灵魂】。

    • 【死胎】

    据产生地而言，魔力场越强，越容易产生意识，魔力场越弱，越容易失败。产生意识失败的大脑此后将停机，产生所说的【死胎】现象。（变成没有意识的躯壳（植物人？））

    • 【游魂】

    在智慧生物的生命终末期，当肉体死亡抑或灵魂遭受重创，在自然状态下，灵魂将四散爆发，变为失去大部分意识的【游魂】。

    • 【借尸还魂】

    如果附近存在寻找躯体的【游魂】，那么【游魂】与【死胎】可以结合。在结合过程中，【游魂】的多余部分将被剔除，仅保留部分记忆重生。

    • 新诠释【游魂的旅途】

    尚能维持存在的游魂在自身意志和魔法粒子分布的双重影响下，可能前往数个目标地：

    1. 不论地点：与【死胎】结合，重生

    2. 不接受死亡，前往魔法粒子富集的魔界：恢复意识成为幽灵/重生

    3. 接受死亡，依照世界的魔法粒子平衡前往魔法粒子稀缺的天界：成为新神

    4. 新诠释【为何散去】

    灵魂的本质是意识的后产物。意识本身会产生痛苦、绝望等情绪，而当这些情绪产生时，我们可以称灵魂出现了【磨损】或是【破碎】现象。当灵魂达到临界点，它将彻底放弃自己的主体性，回归魔法粒子的形态。

    时间对任何个体都是平等的。作为生命在这个世界上度过越多年月，痛苦的袭击也会越发频繁，灵魂的消散不以轮回次数计，以主观情绪的变化计。

    产生怎样的情绪与个人性格经历有关，于是不同的灵魂有不同的消散周期。此外，意识的积极主观性在很大程度上对于保存灵魂的完整性有良好的效果。

    • 【天界人的回收利用】

    天界魔法粒子最为稀薄，因此死胎现象频发。天界人借此人为将从人界前往天界的，失忆的游魂置入死胎，提高存活率。

    天界人对魔界灵魂的利用近乎于零，他们不在乎魔界的灵魂运行机制。

    • 【对亡灵法术战争始末】

    唯一可以影响灵魂的法术就是亡灵法术。或者说，亡灵法术便是对所有影响灵魂的法术的称呼。亡灵法术将存在于人界的游魂置入自己的仆从亡灵，阻断了部分灵魂的输送，间接影响了天界人的存活率。

    采取驱逐人类的方针，乃是出于除之不尽的难题。被驱逐者在魔界建立新的基地，这为其他巫妖提供了前往基地的理由。

    • 【余震】

    在巫妖的不懈努力下，大量骷髅、僵尸、幽灵加入了魔界的主要人口群体，魔界生产力得到了提升。

    1. 亲爱的小天使

    假如，你在轮回后作为一名天使出生，睁开了双眼，你所见的将是一片字面意思的乐园。

    大理石柱撑起庄严的圆顶，没有一丝缺憾的切割技艺和设计技巧呈现出完美的弧形。明艳的花朵在茶几上的陶罐之中盛放，书籍的香气充满房间。你看见头顶有一个好玩的旋转玩具，其上摇晃的吊坠如同天平的形制。

    你有疼爱你的父母——一位背生双翼的优雅绅士，一位同样背生双翼的美丽女士，他们头顶有一圈不显眼的光环。他们穿着干净整洁，并不华丽的衣服，和你极其相似的面容和眼眸饱含慈爱。但他们看向你的眼神在惊喜中同样带着一丝担忧，你尚且不知道为何。

    他们将婴儿床中的你抱起，欢天喜地地说着天界语，这也将是你此后一生使用的母语。这种语言在人界的天教祈祷仪式中被模仿，但不完全一致，这是一种基于表意文字的复杂语言，每个字能够表达相当多的意涵。

    起初，周围的光线令你感到不适——那实在是太温暖、光亮了。然而父母带你来到室外后，你却骤然发现，方才的光线实在是太温柔了，天界白昼的双子太阳散发着无处不至的辐射，不断洒下金黄色恩赐，让整个世界被无限照亮，几乎不留下任何阴影。

    然而即使是你婴儿柔嫩的皮肤，被此阳光直接照射也不会觉得刺痛，因为你总有一天会得知：这双生的太阳实际上是天使智慧的结晶，而非自然。

    天使们在你们一家三口的居所之外集聚，有人拉开了拉炮，有人欢呼起来，所有天使的外形都是那样赏心悦目。每个天使都小心翼翼地观察着被母亲抱着穿过人群的你，他们出于某种原因无条件地喜欢你，这一点你从心底感觉得到。

    丰富的绿色和蓝色植物品种围绕在你们身边，有一位天使站在队列最末，持白色花环款款走来，将花环套在你们母子二人身上，接着双手相握，闭上双眼后将双手抵在额头，她虔诚的祈祷让你不由自主地抬起小手模仿，你的母亲见你乖巧的样子，伸手摸了摸你的头顶，她的手指稍有些冷，不过动作非常轻柔。

    “亲爱的小天使，感谢你们来到我们身边。祝你找到属于你的命运——”

    天使们依次对你施与祝福，淡淡的各色光芒闪现在你们周围，这既是美好的心愿，也是实际生效的“魔法”。

    漫长的一天结束，你觉得有些疲惫，作为婴儿的你即将陷入沉睡，但一件事吸引了你的目光——

    那夜晚没有月光，黑暗包裹了所有一切。

    因为乐园不需要狡诈的月亮，乐园也不存在夜晚出行的理由，所有事物运行理当如律法所言，喇门。

    小天使的成长（表）

    小天使很快便能意识到，自己所在的地方像是一个小村庄，村庄的人口约有二十户，而能够称为孩子的——甚至不是年龄相仿的——其他小天使只有2~3人。

    小天使的成长速度是很快的，与人类相仿。天使有且仅有男女两种性别，至少在小天使见到的世界中是这样的。

    在家庭教育中，父母将会教育小天使如何用餐、如何沐浴、如何获取生活所需的物资，并在十岁后将孩子送去其他城市的天使小学。

    • 用餐：在村庄附近分布着类似小麦和大麦的植物农田，天使们分工进行农作，每位村庄中的天使都会参与至少一个环节的生产，他们所工作的时长大致相同。将麦子收割后简单地加上水煮成可口的麦粥，这就是天使的主食。此外水果之类也在餐桌之上摆放，但村庄中并不出产水果，需要与其他村庄进行交易。

    • 沐浴：村庄中分布着水井一样的设施，但其水的来源不明，因为村庄附近，至少在小天使彻底离开村庄前往小学前都见不到河流的分布。因此用水是受到一定程度的限制的，需要珍惜。另一方面，天使不太需要洗衣，这方面的需求会通过洗衣魔法解决。

    • 生活所需的其他物资：如牙刷、牙膏一类的东西是存在的，虽然供应也仅限于够用，但木质刷柄和铁皮膏皮是存在的。每月，这些东西会通过传送阵由派送员送来。也可以进行订货，但必须每月向配送员提交清单。

    • 货币：小天使不知道什么是货币。

    天使村

    在村中的二十户天使是经过严格计算的，有利于产生和孕育小天使的人员。他们的存在皆是为了产生新的天使，是被天使社会筛选过的最普通、保持基本健康情况、并保持着一定明度的天使。（明度系统后文介绍）

    天使的性别本身没有过多的意义，因为天使的魔法是可以任意改变自己的形体的。但为模仿人界的繁殖系统，天使认为这种两两结对的模式最有利于提高生育率。

    天使的生理结构与人类相差不多，在一般情况下，同样是每年一胎，每胎一个孩子。然而因为魔法粒子的缺乏、魔力场的稀薄，孩子的大脑未必能够产生属于自己的灵魂。

    因此，天使需要游离的灵魂，并将其引入未能产生灵魂的躯壳，提高小天使存活率。——为此，村中的人员必须全都掌握操作灵魂的能力，最低限度也是能够将一个能量尚且充盈的灵魂引入新生儿之中的操作员。

    除了操作灵魂之外一无是处的天使——这样说似乎不太确切，因为操作灵魂正是天使的看家本领。也许可以说，村里的天使都是对收集信仰没有兴趣的人吧。

    2. 天使生产力学说

    对于天使来说，生产某种物件有两种方式：

    1. 种出来
    2. 用魔法搓出来（需要【信仰】与【魔力】）

    信仰是除魂源之外第二种由灵魂产生的资源。

    天使因天界魔力场稀薄而陷入生存危机（第一次大饥荒）时，付出了惨痛的代价，才再次打开了通往人界的高维次元通道（此时的通道还不能通过生物）。

    而打开通道后，便能够从当时还相互认识的部落氏族得到【信仰】，这是出于绝对信任的情绪产生的能量。

    这种能量不能如魂源那样直接作为魔力的代用品使用，它作为一种介质/润滑剂，能帮助天使们从高维次元通道中获取人界的生物借助魔法粒子产生的能量。这种能量被天使们同样称为魔力，但与人界大相径庭的是，这是一种可封装、可即时放出的能量块。

    所以综上所述，流程如下：

    1. 原初世界的一个（或多个）高魔部落氏族被分割到了魔力场稀薄的天界
    2. 天界的自然资源因长期的懒惰化发展，无法满足所有氏族人口的日常需求
    3. 惨烈的大事件（氏族战争）和第一次大饥荒分不清是哪一个先发生，又是哪一个引发了哪一个，总之按照考古，是差不多同期发生的
    4. 某个小型的氏族组织付出惨痛的代价，打开高维次元通道
    5. 通道之中获取到其他相识氏族的【信仰】，并经由润滑效应，携带了可用于释放魔法的魔力到天界
    6. 所有存活的氏族达成共识，共同收取信仰，共同劳作，努力用他们高超的魔法技术填补食物的空缺

    因此，信仰才是尤为重要的资源。信仰能量也可以被储备，但储备需要消耗魔力。因此天使的社会组织需要把握好何时将信仰转化为可用的魔力，而何时应该储备信仰，以供紧急时刻爆一波魔力。

    解决资源紧缺问题的天界人共同订立了【收取信仰、早日回家】的目标。而为了扩大通道，需要大量的魔力。天界人必须在保证族群生存的同时，积攒剩余魔力，必须精打细算。

    对天使来说，魔力的妙用太多，它对第一产业的产量提升远远大于攀爬农业科技树。但另一方面，需要节约魔力，因此对于农业科技他们也有一些基础的加点。这是第一次饥荒解决后确定的方针。

    3. 天使的断代

    1. 世界撕裂
    2. 【大灾变】第一次大饥荒与氏族大战争，惨烈的大事件
    3. 信仰拯救了天使，【瘟疫时代】因死亡到来，在外力压迫下天使达成同盟，借助魔力共克时艰。
    4. 【青苹果时代】稳定发展期，人口保持恒定，生育率问题没有突出显现。双子太阳升起，星空遮蔽
    5. 【黄金世】人口继续保持恒定，魔力迎来结余，生产力初显溢出，人力得以不仅分散到魔法研究和穿界研究，更分散到文学、艺术、工匠等领域，为此后的宗教打下基础。
    6. 【遗忘纪元】从人界来的魔力收入降低，魔力再显疲软。但由于科技树的加点，生产力仍然溢出，魔力利用率不断提高。发现了天界人生存的极限：仍然是肉体凡胎的生物，被杀就会死，虽说魔力治愈能力强，但天使的躯体强度也是有极限的。——开始正式研究死胎现象与关注人界发展。
    7. 【宗教的诞生：天历前2435年】天界终于打开了通往人界的生物通道，但为时已晚。此时人界西陆的三大氏族车得正开心，也很少有人知道他们共同的起源（仅保存在部分巫医或祭祀者手中）。 经过多轮协商、激烈的意见交换，天使内部终于达成了统一意见，以协助天魁氏族答应氏族战争为交换，让天教成为天魁氏族的唯一宗教（天魁氏族首领库迦什应允，后来他又想办法搞定了反天教思潮，在这里可以对正知道历史同源真相的祭祀群体） 但还有一个问题：现在天使的终极目标“回到人界”已经实现了，但他们发现沧海桑田，回了也没用。新的目标是什么，又将走向何处？ 天使领袖订立了新的目标：逐步帮助人界实现跨越式发展，拉齐天界与人界的文化水平后，一同探索合并三界的方法。

    为何不能直接让人界飞升？ 解释历史是长期的问题，而且不利于收集信仰；人类已经变成了下级生物，这是不可逆的，人不可能变成天使；内部反对派的作用等

    1. 【调和年代】在此期间，天使着力祛除巫妖对灵魂的干涉，这是出于稳定天界人口增长率的考虑。
    2. 【新纪元】为让新世代的天使通过天教这一更具效率的信仰机器收割魔力，全面改写历史，给新生者以全新的自我认知，不再将人类视为同胞，而是信徒。

    4. 天使的色彩

    在天使的力量体系中，每位天使有着自己的【色彩】，这是一种基于独特魔力应用体系的人体辉光现象。色彩的释放是不由意志决定的，当天使使用魔法，色彩便会被释放，这是一种与人类可见光谱范围仅稍有交接点的光，部分幼年人类可见，全体天使可见。

    色彩作为一种辉光，每次释放魔法时产生，持续时间可达3小时~5天不等。因此大多数天使都会常时带有色彩。色彩是可以累积的，因此使用魔法越频繁，辉光强度越高。高强度的辉光色彩将会扩大在光谱上覆盖的频率，因此也会为更多人所见。

    光谱之中的颜色是无限多的，每位天使的色彩都是独一无二的。但天使也没有能够明确辨别#456798和#456799的能力。

    熟悉光谱和调色的朋友可能会说，这两个色号是有明暗度的，不是天然色环的颜色。对的，因此我们还要介绍其他系统。

    决定色彩的要素

    1. 色彩与天使长期使用的魔法类别有一定关联，更强大的魔法将会让色彩的饱和度更高。

    2. 色彩与天使的思想倾向有关联：

    3. 红色：社会规律与抽象的社会意志，肯定这些东西并主动维护规则的，认为规则是凌驾于个体之上的
    4. 蓝色：自由意志的拥护者，崇尚变通与变化，极具个性
    5. 绿色：更在乎社会与人之外的东西，比如自然和生存艺术
    6. 红+绿=黄色：具有献身精神的个体，将自我弱化的存在
    7. 红+蓝=品红（粉紫色）：在个人与社会关系之中取得平衡，懂得利用社会的种种机制保护自己的享乐主义者
    8. 绿+蓝=青：漂浮在社会之中，不事生产的、漠视法律存在的。

    对以上思想的认可程度会反映在色彩的纯度这一属性上，越认可，越纯。

    1. 色彩明度：白与黑，是能够覆盖于所有其他色彩的明度属性。
    2. 白：代表【天使全体的利益】
    3. 黑：代表【损害天使全体的利益】
    4. 标准表达方式：10% 白，90%黑。
    5. 对于一个个体，我们说“有明度”即指白色含量超过50%。
    6. 天然存在的天使不会达到0%黑或0%白。可以无限接近。

    色彩带来的影响

    一定程度的思想透明，给天使社会带来了管理上的方便。

    说到底，色彩只是一种辉光现象。因此只要主动使用魔法，将能量转化为光能就能轻松改变自身周围的物理存在的光的颜色，然而光的交叠会导致明度下降。

    但是，天使社会同时也是偏向自由的，天使组织没有严格管理天使的必要，只要能够获取足量的信仰，就会欢迎其存在。

    信仰没有好与坏。它是且只是一种能量。

    其他

    在中层世界的人之中，只有幼年体那尚未发育完全的眼睛才能在天使的色彩较为浅薄时完全分辨其具体颜色。除去泛人类之外的其他生物（包括植物）也能够感知色彩，但感知较为模糊。

    如果天使在中层世界使用魔法（这轻而易举，因为中层世界的环境还要更好），那么可以见到色彩的人群将会扩大。光辉越是明亮，可见人群越大。

    天教将色彩视为神圣的存在，圣光魔法也有着模仿色彩的部分特质——不同类型的圣人会有自己指定的一种色彩，但这只是模仿品。
